# PasteSync SVG Icons

These files are intended for direct GitHub raw links and external embeds.

- `pastesync.svg`: main branded icon with background
- `pastesync-transparent.svg`: transparent mark for dashboards with their own tile background
- `pastesync-homepage.svg`: recommended square icon for `gethomepage`

Example raw URLs after pushing to `main`:

- `https://raw.githubusercontent.com/zewelor/pastesync/main/icons/pastesync.svg`
- `https://raw.githubusercontent.com/zewelor/pastesync/main/icons/pastesync-transparent.svg`
- `https://raw.githubusercontent.com/zewelor/pastesync/main/icons/pastesync-homepage.svg`
